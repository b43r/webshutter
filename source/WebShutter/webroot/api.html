WebShutter API

GPIO API

GET /api/gpio/[pin]/[value]

[pin] : GPIO Pin Nummer
[value] : 0 = Low, 1 = High

Shutter API

GET /api/shutter/[group]/[sequence]

[group] : Rollladengruppe (1-4)
[sequence] : Befehle durch Kommas getrennt (z.B.: dd,5000,u,u,u)

Befehle:

  • dd: Ganz herunterfahren (Puls 600ms)
  • d: Hochfahren stoppen bzw. Lamellen eine Stufe schliessen (Puls 100ms)
  • uu: Ganz hochfahren (Puls 600ms)
  • u: Herunterfahren stoppen bzw. Lamellen eine Stufe öffnen (Puls 100ms)
  • Zahl: Verzögerung in Millisekunden
